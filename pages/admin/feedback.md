# Отзывы

Возможности
- Добавить отзыв
- Удалить отзыв
- Редактировать отзыв

Внешний вид:

Список отзывов:
- Текст отзыва
- НикНайм
- Кнопка удалить отзыв напротив отзыва
- Кнопка редактировать отзыв напротив отзыва


Кнопка добавить отзыв:

Открываеться модальное окно где есть поля:
- НикНайм
- Текст
